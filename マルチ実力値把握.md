

# 見解書（現行アライメントマルチ実力値把握結果）

作成日：2026.5.21

---

# 1. 背景（なぜ実力値把握を行ったのか？）

- 新商品検討会（2026.3.16）にて、
  バンザイ社のアライメントマルチ4tに対抗すべく、
  企画案が上がった。

- ASS2027に向けた検討の一環として、
  現行品であるアライメントマルチ3tについて、
  自社規定のホイールアライメント測定レベル精度を
  満たしているか確認を行うこととした。

- 自社規定では、
  - 前後差：4mm以内（対角含む）
  - 左右差：2mm以内
を規定値としている。
<img width="554" height="480" alt="スクリーンショット 2026-05-27 125721" src="https://github.com/user-attachments/assets/aa5e7b12-91de-4f68-a754-7c4482a921c5" />


- そこで、
  三次元測定機を用いて、
  現行機の実力値把握を実施した。
  　　
# 2. 結論

- 自社規定のレベル精度（前後4mm・左右2mm）に対して、
  実測値は前後4.9mm・左右0.7mmとなり、
  **レベル精度は満たしていない**ことが分かった。

- また、ドライブオンプレートよりも
  リンク側の方が荷重を掛けた際の変位量が大きいことが分かり、
  アライメントマルチ4t構想時には
  **リンク剛性不足になる可能性が高い**。

- そのため、
  レベル精度を出すために
  **リンクの構造変更を推奨する。**

# 4. 結果・考察

## 無負荷を0とした変位量結果（床面からの高さ）
<img width="854" height="335" alt="スクリーンショット 2026-05-27 130111" src="https://github.com/user-attachments/assets/c877cf02-73a5-461b-ad04-8a474b79fbe4" />

## 結果

- 1.ドライブオンもリンク支点もF側よりもR側の方が変位量が大きい。
- 2.F側とR側それぞれで、
    ドライブオンとリンク支点の変位量はほぼ同じ。

---

## 考察

- 1.降下止めツメ支持がF側のみの構造となっているため、
    F側に対してR側の剛性が低く、
    R側へ傾きやすい。

- 2.変位の主な原因としてはドライブオン自体のたわみではなく、
    リンク支点側のたわみの可能性が高い。
